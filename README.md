VS Code扩展，统计文档字数

# 克隆仓库 & 编译 / clone repository & build

```
git clone https://github.com/leqijishu/WordsCount.git
cd WordsCount
npm ci          # 或 npm install
npm run build   # 生成 dist/wordscount-*.vsix
```
